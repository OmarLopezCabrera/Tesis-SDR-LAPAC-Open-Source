# Diseño de una Radio Definida por Software

_Esta tesis describe el diseño del hardware para una Radio Definida por software (SDR). Partiendo de una arquitectura  modular genérica, pasando  a una arquitectura con componentes definidos,  hasta el desarrollo del PCB de la misma.
Esta SDR cubre un rango de frecuencia de 70 MHz hasta 6 GHz, tiene una conexión USB 3.0 y una FPGA Artix-7. Dicha FPGA puede ser programable por USB; esta característica permite realizar experimentos en un rango de las siguientes señales: FM, TV brodcast, celular, Wi-Fi entre otros. El Front End de RF está compuesto por el transceiver AD9364 de Analog Devices el cual tiene 56 MHz de ancho de banda instantáneo.
